## 2.Algriothm ##
#### 2.1 第一部分 ####
    第一部分实验要求实现run_global_transform.py脚本，实现伸缩、平移、旋转等基本图像变换功能。具体实现如下：
    1. 读取输入图像。
    2. 根据输入参数，计算图像的伸缩、平移、旋转等变换矩阵。其中，伸缩矩阵为缩放因子，平移矩阵为平移向量，旋转矩阵为旋转角度。
        2.1 平移：通过构造矩阵[1, 0, tx; 0, 1, ty; 0, 0, 1]实现；
        2.2 旋转：通过构造矩阵[cos(θ), -sin(θ), 0; sin(θ), cos(θ), 0; 0, 0, 1]实现；    
        2.3 缩放：通过构造矩阵[sx, 0, 0; 0, sy, 0; 0, 0, 1]实现；
    3. 根据变换矩阵，对图像进行变换。具体实现为，将图像的每个像素点的坐标乘以变换矩阵，得到新的坐标，然后将新的坐标映射到原图像上，得到变换后的图像。
    4. 将变换后的图像保存为输出文件。



#### 2.2 第二部分 ####
    第二部分实验要求通过移动最小二乘法(Moving Least square)方法或者RBF方法实现基于点的图像变形功能。具体实现如下：
    1. 读取输入图像，并转换为灰度图像。
    2. 根据输入参数，计算图像的伸缩、平移、旋转等变换矩阵。其中，伸缩矩阵为缩放因子，平移矩阵为平移向量，旋转矩阵为旋转角度。
    3. 根据变换矩阵，对图像进行变换。具体实现为，将图像的每个像素点的坐标乘以变换矩阵，得到新的坐标，然后将新的坐标映射到原图像上，得到变换后的图像。
