# 实验描述

本实验通过迭代优化的方式实现了图像的``Possion``融合。

## 实验原理

传统的图像剪贴会产生生硬的边缘，因为源图像``（Source）``和目标图像``（Target）``的背景色调不一致。

``Possion融合``的核心思想是：**“保持梯度（变化趋势）比保持像素值更重要”。**

本次实验中不直接复制像素，而是解一个方程，使得融合区域的二阶梯度（拉普拉斯算子$△$）尽可能接近源图像的梯度。这样，图像的内容（高频信息）被保留，而色调（低频信息）会自然地与背景平滑过渡。

## 代码实现 

### 损失函数

本实验补齐了``cal_laplacian_loss``函数，该函数利用卷积核``$[[0,-1,0], [-1,4,-1], [0,-1,0]]$``提取源图像与当前结果图的Laplace特征，计算两者间的MSE。
