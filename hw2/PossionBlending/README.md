# 实验描述

本实验通过迭代优化的方式实现了图像的``Possion``融合。

## 实验原理

传统的图像剪贴会产生生硬的边缘，因为源图像``（Source）``和目标图像``（Target）``的背景色调不一致。

``Possion融合``的核心思想是：**“保持梯度（变化趋势）比保持像素值更重要”。**

本次实验中不直接复制像素，而是解一个方程，使得融合区域的二阶梯度（拉普拉斯算子$△$）尽可能接近源图像的梯度。这样，图像的内容（高频信息）被保留，而色调（低频信息）会自然地与背景平滑过渡。

## 代码实现 
本项目采用了一种基于 ``自动求导优化 (Optimization-based)`` 的泊松融合实现方案，相比传统的稀疏矩阵求解器（如 scipy.sparse.linalg），该方案在 GPU 加速下具有更高的效率和更好的交互灵活性。

### 损失函数

本实验补齐了``cal_laplacian_loss``函数，该函数利用卷积核``$[[0,-1,0], [-1,4,-1], [0,-1,0]]$``提取源图像与当前结果图的Laplace特征，计算两者间的MSE。

### 构造掩码

在交互式编辑中，如何从用户不规则的点击中生成高质量的掩码（Mask）是第一步。本实验在此函数中实现了稳健的几何转换逻辑：

1. 多边形填充算法：利用 ``cv2.fillPoly`` 将用户点击的离散点集（Points）转换为闭合的多边形区域。
  
2. 坐标鲁棒性：函数显式地将输入点转换为 np.int32，有效防止了坐标偏移或浮点数导致的边界锯齿问题。

3. 二值化表征：采用 0/255 的标准二值格式，为后续 PyTorch 张量转换和 torch.where 的像素索引提供了精准的逻辑支持。

```
python run_blending_gradio.py
```
